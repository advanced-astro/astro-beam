## Install

```sh
bun install astro-beam
```

```sh
npm install astro-beam
```

```sh
pnpm install astro-beam
```

```sh
yarn add astro-beam
```

## Usage

```astro
---
import { BeamAnalytics } from "astro-beam";
---

<BeamAnalytics token="11111111-2222-3333-4444-555555555555" />
```

with [`Partytown`](https://partytown.builder.io) lazy loading

```astro
...

<BeamAnalytics token="11111111-2222-3333-4444-555555555555" partytown=true />
```

or short

```astro
...

<BeamAnalytics token="11111111-2222-3333-4444-555555555555" partytown />
```
